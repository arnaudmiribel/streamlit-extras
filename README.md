## Getting Started

### Installation

```
pip install streamlit-extras
```

### Usage

<sup>Visit <a href="extras.streamlitapp.com/Stoggle">extras.streamlitapp.com</a></sup>
```python
import streamlit_extras as stx

stx.stoggle("Summary", "More content that's hidden by default!")
```
